# `tag`

Used for marking and selection

## Example

### Basic usage

```html
<tag text="'tag 1'"></tag>
<tag text="'hello'" color="#ff0000" theme="Error"></tag>
```

### Closeable

```html
<tag text="'hello'" closeable></tag>
```
