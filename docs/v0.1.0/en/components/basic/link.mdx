# `link`

Used to link to an address, with an underline effect by default.

## Example

### Basic usage

```html
<link theme="Info" text="'Github'" href="'https://www.github.com'"></link>
```

### Button link

```html
<link
    theme="Info"
    text="'Github'"
    href="'https://www.github.com'"
    padding="10.0 16.0"
    border_radius="2.0"
    underline_visible="false"
    background_visible="true">
</link>
```
