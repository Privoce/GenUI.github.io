# Built-in types

In this document, we provide all the built-in types that you can use directly in the `GenUI` project. 
Generally speaking, you will rarely use these types because `GenUI` has shielded the lowest-level component construction for you.

However, since the writing of the attribute values ​​of these built-in types will be involved in the built-in components, 
we still provide you with a relatively complete writing reference.

> [!TIP]
>
> If you have studied the value parsing and writing in the [Data Type](/doc/tutorial/data) section, 
> you should be able to understand how to write it very easily.

## `Align`

### Usage

range: `[0.0, 1.0]`

- `{x: 0.5}`
- `{y: 0.2}`
- `{x: 0.2, y: 0.9}`

### Source Code

```rust
pub struct Align {
    pub x: f64,
    pub y: f64,
}
```

## `Padding`

### Usage

- `1.0` means: `{left: 1.0, top: 1.0, right: 1.0, bottom: 1.0}`
- `1.0 2.0` means: `{left: 2.0, top: 1.0, right: 2.0, bottom: 1.0}`
- `1.0 2.0 4.0 6.0` means: `{left: 6.0, top: 1.0, right: 2.0, bottom: 4.0}`

### Source Code

```rust
pub struct Padding {
    pub left: f64,
    pub top: f64,
    pub right: f64,
    pub bottom: f64,
}
```
