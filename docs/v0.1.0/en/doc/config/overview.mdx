## Type mapping

In most GenUI projects, `toml` is used for configuration, and `Rust` types are used to map `toml` to types.

| Rust             | TOML                    | TOML FORMATTER       |
| ---------------- | ----------------------- | -------------------- |
| `String`         | `String`                | `"  "`               |
| `i_type`         | `Integer`               | `0`                  |
| `u_type`         | `Integer`               | `0`                  |
| `f32, f64`       | `Float`                 | `0.0`                |
| `bool`           | `Bool`                  | `true, false`        |
| `Vec<R>`         | `Array<T>`              | `[  ]`               |
| `enum (United)`  | `String`                | `"  "`               |
| `enum (Named)`   | `Table` / `InlineTable` | `[[struct name]] {}` |
| `enum (Unnamed)` | `Table` / `InlineTable` | `[[struct name]] {}` |
| `struct`         | `Table` / `InlineTable` | `[[struct name]] {}` |

> [!TIP]
>
> `String`always use `snake name rule`
