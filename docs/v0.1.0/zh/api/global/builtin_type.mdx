# 内置类型

在本文档中我们提供所有您可以直接在`GenUI`项目中使用的内置类型，通常来说您很少会用到这些类型，因为`GenUI`已经帮您屏蔽了最底层的组件构建。

但由于在内置组件中会涉及到这些内置类型的属性值的写法，所以我们依然提供给您较为完整的写法参考。

> [!TIP]
>
> 如果您学习过[数据类型](zh/doc/tutorial/data)部分的值解析与写法，您应该非常容易理解如何编写。

## `Align`

### 写法

取值范围: `[0.0, 1.0]`

- `{x: 0.5}`
- `{y: 0.2}`
- `{x: 0.2, y: 0.9}`

### 源码

```rust
pub struct Align {
    pub x: f64,
    pub y: f64,
}
```

## `Padding`

### 写法

- `1.0` 表示: `{left: 1.0, top: 1.0, right: 1.0, bottom: 1.0}`
- `1.0 2.0` 表示: `{left: 2.0, top: 1.0, right: 2.0, bottom: 1.0}`
- `1.0 2.0 4.0 6.0` 表示: `{left: 6.0, top: 1.0, right: 2.0, bottom: 4.0}`

### 源码

```rust
pub struct Padding {
    pub left: f64,
    pub top: f64,
    pub right: f64,
    pub bottom: f64,
}
```

## `Size`

### 写法

- `Fit`
- `Fill`
- `200.0` 表示: `Size::Fixed(200.0)`
- `Fixed(200.0)`

### 源码

```rust
pub enum Size {
    #[default]
    /// Fill the size of the parent widget
    Fill,
    /// detail size of the current widget
    Fixed(f64),
    /// Fit content
    Fit,
    All,
}
```
