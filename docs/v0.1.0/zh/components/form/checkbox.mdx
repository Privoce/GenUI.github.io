# `checkbox` 多选框

我们推荐您将`checkbox`放在`checkbox_group`中进行使用

> `checkbox_group`继承自`view`

## 示例

### 基础使用

如果要表示选中状态，使用`selected`进行指定

```html
<checkbox_group>
    <checkbox text="'rap'"></checkbox>
    <checkbox text="'play basketball'" selected="true"></checkbox>
</checkbox_group>
```

## 事件回调

### `checkbox`事件回调

| 名称                  | 说明            | 参数              |
|-----------------------|-----------------|-------------------|
| `clicked`            | 点击事件         | `GCheckboxClickedParam`|
| `hover_in`           | 鼠标悬停事件     | `GCheckboxHoverParam` |
| `hover_out`          | 鼠标移出事件     | `GCheckboxHoverParam` |

### `checkbox_group`事件回调

| 名称                  | 说明            | 参数              |
|-----------------------|-----------------|-------------------|
| `changed`            | 状态改变事件     | `GCheckboxGroupEventParam`|
