# `input` 输入框

通过鼠标或键盘输入字符

> [!NOTE]
>
> 目前输入框的能力较为单一，我们预计在下个版本推出更多或更强大的输入框

## 示例

### 基础使用

```html
<input
    text="'Hello'"
    height="36.0"
    width="200.0">
</input>
```

## 事件回调

| 名称                | 说明             | 参数                 |
| ------------------- | ---------------- | -------------------- |
| `changed`           | 输入值变更事件   | `GInputChangedParam` |
| `keydown_unhandled` | 未处理键按下事件 | `KeyEvent`           |
| `escaped`           | 按下 Esc 键事件  | `KeyEvent`           |
| `key_focus`         | 获得键盘焦点事件 | `KeyFocusEvent`      |
| `key_focus_lost`    | 失去键盘焦点事件 | `KeyFocusEvent`      |
