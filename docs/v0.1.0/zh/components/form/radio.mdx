# `radio` 单选框

我们推荐您将`radio`放在`radio_group`中进行使用

> `radio_group`继承自`view`

## 示例

### 基础使用

您可能发现了，`radio_group`中有`selected`而`checkbox_group`中没有，这是当前的一个需要额外注意到点，我们会在下个版本中统一最终的风格

> [!WARNING]
>
> 在`radio`中设置`selected`是个不明智的做法，因此当您使用`radio_group`包裹时最终编译可能无法通过
>
> 特别是如果您同时设置了多个`radio`都是`selected="true"`

```html
<radio_group selected="1">
    <radio text="'male'"></radio>
    <radio text="'female'"></radio>
</radio_group>
```
