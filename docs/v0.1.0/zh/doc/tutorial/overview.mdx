## 学习顺序推荐

<Steps>
    ### QuickStart

    > GenUI快速入门项目

    ### 项目结构说明

    > 解释GenUI项目的结构和各个文件的用途

    ### `<template>`

    > 模版结构语法部分，GenUI `template` 采用类`HTML`的书写方式并在此基础上进行了扩展与约束

    ### `<style>`

    > 模版样式语法部分，GenUI `style` 同样是类原生`CSS`的并进行了相关的增强，它的书写方式更趋近于`SCSS`
    > 
    > 此外`style`也可称为`prop`，因为它对应了`template`中组件的属性

    ### `script` (API)

    > GenUI的"脚本"部分，使用rust语言，虽然称之为脚本，但和典型js，py，lua不同

</Steps>
