# 主题

`GenUI Theme Color` 是 `GenUI` 的主题颜色，包括一系列用于定义组件颜色的颜色变量。我们提供了默认的配色方案。

在颜色类型上，我们主要分为以下几种：

1. Dark
2. Primary
3. Info
4. Success
5. Warning
6. Error

每种颜色类型都含有 11 种色值: `[25, 50, 100, 200, 300, 400, 500, 600, 700, 800, 900]`

**其中主色值为`500`**
