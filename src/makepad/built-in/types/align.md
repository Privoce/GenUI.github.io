# Align

The `Align` struct defines alignment values along the x and y axes.

## Properties
|decorate|name|type|description|
|--|--|--|--|
|live| x| `f64`| The alignment value along the x-axis. |
|live| y| `f64`| The alignment value along the y-axis. |

## Example

```rust
align: {x: 5, y: 6.0},
align: 0.5,
```
